FixItUp

FixItUp is a step-by-step repair guide app for common household problems. Users describe what's broken — a running toilet, a stuck drawer, a flickering light — and FixItUp walks them through the repair with illustrated instructions, tool lists, and video clips. No more paying $200 for a plumber to tighten a valve.

The app uses a simple diagnostic flow: answer a few questions, and it narrows down the likely issue and serves up the right guide. Built with React and a custom CMS for the repair database.
