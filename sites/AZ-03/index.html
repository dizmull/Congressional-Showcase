// About Me

I'm a senior at Mountain Pointe High School in Phoenix, Arizona. I've been making beats since 6th grade when my older brother left his MIDI controller at home and never asked for it back. Since then I've produced over 200 tracks, built a following on SoundCloud, and even sold a few beats to local rappers.

I taught myself Python and JavaScript along the way because I got tired of using other people's tools and wanted to build my own. BeatDrop is the result of that energy.

// BeatDrop

BeatDrop is a collaborative beat-making web app where you and your friends can build tracks together in real-time. Think Google Docs but for music. You get a 16-step sequencer, a library of drum kits and synth sounds, and a chat panel so you can vibe with your collaborators.

I built it using the Web Audio API, WebSockets for real-time sync, and React on the front end. The craziest part is that two people in completely different cities can lay down a beat together with almost no lag. My friends and I use it literally every weekend.
