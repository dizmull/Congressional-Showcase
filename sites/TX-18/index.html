LinguaFlip

LinguaFlip is a free flashcard-based language learning app with a twist: instead of just memorizing words, you learn through real conversations and cultural context. Each card includes pronunciation audio, example sentences, and a fun fact about the culture behind the language.

Right now it supports Spanish-English and English-Spanish, but I designed the backend so volunteers can add any language pair. It's built with React and uses spaced repetition algorithms so you actually remember what you learn!
