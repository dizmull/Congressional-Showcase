# Participant Page Template

This is your starter template for creating your showcase page.

## How to Use

1. Copy this `template/` folder to `sites/XX-00/` (use your congressional district as the folder name, like `CA-12` or `NY-10`)
2. Open `index.html` and replace all the `TODO` sections with your own info
3. Add your entry to `projects.json` in the root of the repo
4. Submit a pull request

See the main [README](../README.md) for full step-by-step instructions.

## Making It Your Own

- **Colors**: Use the built-in Hack Club colors (`var(--hc-red)`, `var(--hc-green)`, etc.) or pick your own
- **Layout**: Rearrange the HTML however you want — the template is just a starting point
- **Images**: Create an `assets/` subfolder in your page folder for any images you want to add
- **Video**: Use a YouTube or Vimeo embed (don't upload video files to the repo)

## Important — Don't Remove These

Your page needs these two lines to work properly with the rest of the showcase:

- The `<link>` to `shared.css` near the top — this styles the navigation arrows
- The `<script>` tag for `nav.js` at the bottom — this enables arrow key navigation between pages

Everything else is fair game to change!
